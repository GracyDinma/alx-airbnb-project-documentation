Translating use case diagram interactions into user stories
- As a guest, I want to be able to login to my account so that I can be able to update my profile.
- As a guest, I want to be able to login to my account so that I can be able to book a property for specified dates.
- As a guest, I want to login to my account so that I can be able to cancel booking based on cancellation policy.
- As a host, I want to be able to login to my account so that I can add and provide property details.
- As a host, I want to login to my account so that I can be able to update properties.
